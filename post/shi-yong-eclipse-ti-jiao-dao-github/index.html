使用eclipse提交到GitHub

2021-01-26 2 min read

一、GITHub创建一个新的repository (Start a project)，复制地址备用。https://github.com/xxxx/TestDemo.git

注：红色标识部分不要勾选，否则后续提交代码时会出现冲突。

二、Eclipse：需要先通过Commit 传到本地仓库，然后再从本地仓库push到Git。
注：Eclipse 高级版本自带Git，不需要安装插件。（如使用低级版本需安装Git插件）。

1、右键项目， Team -> Share Project（共享项目）。

2、勾选上面的，之后Creat Repository，确认选中git。

在此目录下会生成一个.git 的文件夹， 此时Eclipse中文件显示？号。

3、为新建的文件添加Index， 右键项目-＞Team->Add to Index, 这里将所有文件会带个+号， 对于不需要提交的文件，可以通过右键->Team->Remove from Index 移除。

4、首先提交代码到本地仓库
右键项目->Team->Commit, add commit message, 如果遇到下面这种情况，先去Window->Preference->Team->Git->Committing, 将第一个默认勾选去掉。

5、从本地代码提交到Git：项目右键，Team->Remote->Push；

输入Git地址，一般下面的都自动填好了；

点击Next，Source ref和Destination ref下拉框中选择master，点击Add Spec，点击next，上传成功。
